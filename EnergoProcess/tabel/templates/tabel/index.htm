Инструкция по печати табеля бригады

  • Выберите пункт меню "Табели бригад"
  • В меню слева выберите руководителя бригады, для которой нужно сформировать табель
  • В меню слева выберите год, для которго нужно сформировать табель
  • В меню слева выберите номер месяца, для которго нужно сформировать табель
  • Нажмите кнопку "Вывести"
  • Распечатейте табель используя функцию печати браузера (Ctrl-P)

Инструкция по первоначальному заполнению табеля бригады на конкретный месяц значениями по умолчанию

  • При отсутствии записей в табеле бригады на выбранный месяц в момент открытия табеля для печати согласно предыдущему пункту будет выполнено автоматическое заполнение месяца для всех членов бригады типовыми значениями

Инструкция по вводу данных табеля бригады

  • Выберите пункт меню "Редактировать данные"
  • Введите свой логин и пароль
  • Выберите пункт "Табель рабочего времени"
  • Для добавления записи нажмите "Добавить табель рабочего времени" в правом верхнем углу
  • Заполните данные и нажмите "Сохранить" для последующего выхода из режима редактирования или "Сохранить и добавить другой объект" или "Сохранить и продолжить редактирование"
  • Для редактирования записи нажмите на дату нужной записи для перехода в режим редактирования
  • Для редактирования сновных полей достаточно внести изменение в это поле и нажать кнопку "Сохранить" внизу справа
  • Для удобства поиска нужной записи служит фильтр по времени, руководителю или члену бригады в правой части окна
  • Для перехода в режим печати нужно нажать ссылку "Открыть сайт" в правой верхней чати окна
